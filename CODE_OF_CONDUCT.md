# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — issues, pull requests, discussions, or any other contribution — you agree to abide by its terms.

## Reporting

Concerns about conduct can be raised via a [private security advisory](https://github.com/tanaphonble/soulcode/security/advisories/new) (the same channel used for security reports — it is private to the maintainers). Reports are confidential.

The full text of the Contributor Covenant, including the enforcement guidelines maintainers follow, is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
